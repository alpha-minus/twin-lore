# TwinLore — Open Beta

**Seamless Second-screen Multi-tasking for Android Retro Handhelds**

![beta-2](https://github.com/user-attachments/assets/9e18ea9a-a89f-4bf0-af7a-60fa0fd76c55)

TwinLore keeps guides, maps, images, and bookmarks directly tied to the game you are playing. No tab switching. No lost progress. No hunting for last-read pages. Optimized for **controller use on two-screen handhelds**, it maximizes focus and efficiency with style.

**Open Beta** — beta license included, no purchase or setup required. **Ad-free and offline-capable.**

---

## Why TwinLore

The **AYN Thor** enables multitasking, but there’s no single place to organize all game-related content.

After discovering DS/3DS games for the first time, I wanted every emulator or console game to feel that same seamless connection between gameplay and reference material. TwinLore addresses common frustrations:

* Switching constantly between browser tabs or apps
* One-tap Map ↔ Guide switching
* Retro games with no map marking system
* Losing track of previously searched guides
* Forgetting where you left off in a guide
* Managing maps, bookmarks, and media efficiently across multiple devices
* **Track RetroAchivements**

## Key Features

### Automatic Game Detection

* Automatically switch to the currently launched game
* Full support for ES-DE
* Other frontend launchers supported when using RetroArch cores

### Scroll Markers

* Save scroll positions on any online/offline page
* Jump between markers instantly
* Navigate using L1 / L2 or the in-app menu

### Smart Online Search

* Search selected text with the game name auto-appended
* Direct YouTube search from selected text

### Search Templates (Game Landing Page)

* Built-in shortcuts for maps and guides
* Fully custom user-defined search templates
* Open guides or websites with minimal or no typing

### Local Wi-Fi Library Sharing

* Sync your library to another Android device over local Wi-Fi
* Useful for docked or multi-device setups

### Lightweight GameFAQs Reader

* Improved readability for text-based walkthroughs
* Feature Table of Contents for compatible web pages

  * Works online/offline for saved bookmarks
  * For pages without structure, behaves like a quick navigation scrollbar
* Scroll markers can be removed by tapping highlighted text

### Achievements

* RetroAchievements integration

  * Track game achievements in-app
  * Online search for tips per achievement
  * Achievement Grid View with full-tile logos for visual progress tracking
* Works alongside RetroArch and ES-DE detection

### Bookmarks

* Save webpages (online/offline)
* Track reading position automatically
* Per-game bookmark separation

### System & UX

* Standalone, launcher-agnostic
* Controller-first design with shortcuts
* Customizable layouts
* External storage support
* One-button second-display sleep
* LED Accent Support: Thor's LEDs adapt to the game’s artwork color and indicate the active screen
* Works offline (after initial activation), no ads, no tracking
